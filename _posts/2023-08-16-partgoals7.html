Delmål 1: Opnå viden om hvordan SPA og backend kan kommunikere, ved at lave backend klar til at modtage kald fra SPA, og gemme det i sql databasn og returnere data.

Jeg fik lavet en ny consumer og endpoint til API. Jeg og det andet gruppemedlem der står for dette backend delen af vores projekt, samarbejdede om at opsætte backend til SPA. Vi købte en server og lagde API og microservices op på denne, så webapplikationen kan sende anmodninger til den. Jeg fik indsigt i gruppemedlemmets arbejde mht. backend/API, hvilket var godt i forehold til at få et bedre overblik over vores projekt. Jeg lærte desuden lidt om hvordan man opsætter en server. Dog var der en del fejl der skulle rettes mht. API ‘en og opsætning af serveren…


Delmål 2: Opnå viden om hvordan SPA kan lave requests og modtage response fra API, ved at omskrive React koden der håndterer kald til API, så den bruger den nye API gateway i stedet for Google Firebase Storage.

Jeg fik skiftet koden i SPA’en til at lave anmodninger til API gateway, men har ikke skiftet om til at sende til proxy. Jeg lærte om hvordan man strukturer data i React, der skulle sendes til API gatewayen. Dog var det sværere end forventet at omdanne koden I webapplikationen


Delmål 3: Opnå viden om custom hooks i React og skrive om det. Måske, hvis jeg har tid og hvis det giver mening, også implementere det i webapplikationen.

Jeg havde desværre ikke tid til at lære om custom hooks, det kommer til at være i de næste delmål. Pga. tidsmangel måtte jeg prioritere det praktiske, dvs. at få SPA til at kommunikere med API gateway, højere end dette delmål.


Delmål 4: Opnå viden om neurale netværk, ved at læse kapitel 11 i bogen "neural networks from scratch in python".

Jeg havde desværre ikke tid til at læse dette kapitel, det kommer til at være i de næste delmål. Jeg fik mere praktisk viden ved at lave det andet delmål mht. ML, men fik ikke læst dette kapitel. Jeg havde lidt for mange delmål i dette sprint, hvilket desværre medførte at jeg prioriterede andre mål højere.


Delmål 5: Opnå viden om routing i React, ved at lave en side til machine learning forudsigelse i SPA, hvor routing bruges.

Jeg fik lavet routing til siden til ML, og jeg lærte og skrev en smule om routing i React. Men jeg kunne godt have sat mig dybere ind i emnet, og jeg kunne også tænke mig at lave routing med parametre i URL’en til ML siden.


Delmål 6: Opnå viden om og implementere API i Python Flask til ML model

Jeg fik lavet en API til at modtage kald fra ML siden i SPA’en. Jeg lærte hvordan man laver en API i Flask i Python. Det var faktisk ganske nemt at lave denne API.


Delmål 7: Opnå viden om image recognition mht. ML, ved at lave en "proof of concept" image recognition ML model til lækage detection. Så man kan give modellen et billede og den kan sige om der er en lækage eller ej.

Jeg fik trænet en model på termiske billeder, der kan sige om der er en lækage på billedet. Jeg lærte om hvordan man præprocesserer billede data og bedst træner sådan et netværk med tf.keras. Kunne godt have brugt et bedre billede datasæt. Jeg kunne også have brugt mere tid på at sætte mig ind i teorien om dette emne. Men det var et spæmdende experiment og er glad for at jeg gjorde det.
